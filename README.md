# Little Frame

A browser-only editor for making a photo frame with an optional note and up to eight draggable emoji decorations. Exports a transparent PNG ready to share.

## Use

Open `index.html`, choose an image, add up to 8 emojis, write a note if you want one, then select **Export transparent PNG**.

## Publish with GitHub Pages

1. Create a GitHub repository and push these files to its `main` branch.
2. In the repository, open **Settings > Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**, then select `main` and `/(root)`.
4. GitHub will show the public site URL after it deploys.
